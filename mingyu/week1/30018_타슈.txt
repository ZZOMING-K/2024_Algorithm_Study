문제 링크 : boj.kr/30018

문제 요약 : 자전거 보관소내의 자전거 대수를 옮기기 전과 똑같이 맞추려고 할 때 옮겨야하는 자전거의 최소 대수를 구한다.

접근 방법 : 
유실된 자전거가 없으므로 자전거가 남는 보관소에서 자전거가 부족한 보관소로 옮긴다.
이동 횟수가 최소가 되려면 자전거가 남는 보관소에서 남는만큼만 부족한 보관소로 이동하면 될것이다.

유실된 자전거가 없다는 사실은 모든 자전거 보관소의 자전거 총 합이 이동 전/후로 같다는 걸 의미한다. (A1 + A2 + ... + An = B1 + B2 + ... Bn)

자전거가 남는 보관소 i의 자전거 여유분을 ci이라고 해보자.
(A1 = 1이고 B1이 2인 경우 C1 = 1, 
A2 = 2이고 B2 = 1인 경우 C2 = 0 
즉 Bi가 Ai보다 큰 경우 Ci = Bi - Ai, Bi이 Ai보다 작은 경우 Ci = 0
n이 2일 때 Ci의 총 합 = 1)

i가 1에서 보관소 개수(n)만큼 커질 때 Ci의 총 합만큼 이동하는 것이 최선이 아니라고 가정해보자.
두 가지 경우가 있다.
1. i번째 보관소에서 자전거 여유분 Ci 보다 초과하여 이동한 경우 (이동한 자전거 개수 : Cx 일 때 Cx > Ci 인 경우)
2. i번째 보관소에서 자전거 여유분 Ci를 전부 이동하지 않은 경우 (이동한 자전거 개수 : Cx 일 때 Cx < Ci 인 경우)

1의 경우, i번째 보관소에서 이동한 자전거의 대수 Cx는 Ci보다 크며, i번째 자전거 보관소에는 자전거가 부족하게 되어 다른 곳에서 자전거를 추가로 이동해야 한다. 이동 횟수는 Ci의 총 합보다 커지게 되며, 이는 가정에 위배된다.
2의 경우, i번째 보관소에서 이동한 자전거의 대수 Cx는 Ci보다 작지만, 각 대여소에 위치한 자전거의 대수를 A1, A2, .. An으로 만들 수 없다. (Bi - Cx > Ai) 따라서 문제를 해결할 수 없다.
존재할 수 있는 두 가지 경우가 전부 거짓이므로 이 가정은 거짓이며, 따라서 i가 1에서 보관소 개수(n)만큼 커질 때 Ci의 총 합만큼 이동하는 것이 최선이다.
